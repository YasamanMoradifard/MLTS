# Machine Learning for Time Series (MLTS)

I'm bored, and in this repository, I'm challenging myself wiht a combination of **technical skills**, **mathematical engagement**, and **applied coding practice** in machine learning for sequential data.

More professional Blah Blah Blah: 

A structured course companion for **Machine Learning for Time Series (MLTS / MLTS-Deluxe)** at **Friedrich-Alexander-Universität Erlangen-Nürnberg (FAU)**, developed as part of my personal revision on lecture material and hands-on practice in time-series machine learning.

This repository combines:
- a concise lecture summary of the core course topics,
- practical Jupyter notebooks for implementation and experimentation,
- and additional self-practice exercises designed with **ClaudeAI** support.

The goal of this repository is to document both **theoretical understanding** and **practical implementation skills** across classical and modern machine learning methods for time-series data.

## Course Context

This work is based on the university lecture:

- **Course:** Machine Learning for Time Series (MLTS)
- **Institution:** Friedrich-Alexander-Universität Erlangen-Nürnberg (FAU)
- **Lab:** Machine Learning and Data Analytics (MaD) Lab
- **Lecturers:** Dr. Dario Zanca and Prof. Dr. Björn Eskofier

The course covers 14 lectures and progresses from statistical foundations to modern deep learning approaches for temporal data, including uncertainty-aware models, dynamical systems, and sequence modelling. The summary PDF in this repository was written as a personal reference to consolidate the main concepts, equations, intuitions, and practical takeaways from the course.

## Topics Covered

1. Time-series fundamentals, stationarity, and ergodicity, `L01_time_series_fundamentals.ipynb` 
2. Regression and forecasting basics
3. Bayesian inference
4. Gaussian process regression and classification
5. State-space models, Kalman filtering, EKF, UKF
6. Sequential Monte Carlo and particle filtering
7. AR, MA, ARMA, and ARIMA models
8. Time-series data mining, DTW, and matrix profiles
9. Recurrent neural networks, LSTM, and GRU
10. Convolutional and temporal convolutional models
11. Attention mechanisms and Transformers
12. Domain adaptation and out-of-distribution generalisation
